# LeetCode Mastery: Algorithm & Data Structure Solutions

This repository contains my solutions to LeetCode problems, documented with a focus on optimal time/space complexity and clean implementation. As a student at NUS with a focus on high-performance computing and Quantitative Finance, I prioritize C++ for its efficiency, while occasionally using Python for rapid prototyping.

## 🚀 Repository Goals
* **Optimization:** Achieving $O(1)$ or $O(\log n)$ space wherever possible.
* **Consistency:** Daily practice to maintain proficiency in competitive programming and technical interviews.
* **Documentation:** Not just the code, but the *intuition* behind the solution.
